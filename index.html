Supervaize Hello World

This is the hello world project for Supervaize - a FastAPI application demonstrating how to build and deploy agents with Supervaize.

To get started locally, set up your virtual environment with uv venv, install dependencies with uv pip install -e ., and start the server with supervaizer start. For more information, visit the Supervaize documentation.
